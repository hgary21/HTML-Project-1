What is Unity HDRP?

High Definition Render Pipeline (HDRP) is a new rendering system for the Unity Engine functioning under its Scripting Render Pipeline (SRP). SRP allows the user to fully customize the rendering process, from clearing the background, culling the objects, to custom effects during post processing or custom passes. HDRP omits the need for a professional graphics programmer since it provides implemented tools on top of the SRP platform and produces physics based rendering with great graphics while maintaining decent performance.
