HDRP Custom Pass

Custom Pass allows multiple injection points during the render process to execute custom shader code, which can be applied to either fullscreen, procedrual triangles, or any renderers. The different injection points are listed below:

  • BeforeRendering
  • AfterOpaqueDepthAndNormal
  • BeforePreRefraction
  • BeforeTransparent
  • BeforePostProcess
  • AfterPostProcess
